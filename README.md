# 🎓 E-Learning Platform

The **E-Learning Platform** is a web-based learning management system developed using **PHP, MySQL, HTML, CSS, and JavaScript**.  
It allows users to access learning resources, manage courses, and interact with educational content through an easy-to-use interface.

This project demonstrates **full-stack web development**, database connectivity, and user authentication.

---

## 🚀 Features

- 👤 User Registration & Login
- 📚 Course Management
- 📖 Access Learning Materials
- 🧑‍💻 Student Dashboard
- 🗂️ Course Content Management
- 🔐 Secure Authentication System
- 🗄️ MySQL Database Integration

## ⚙️ Installation Guide

### 1️⃣ Clone the repository
git clone https://github.com/chethankumarsv/ELearning-Platform.git

### 2️⃣ Move the project

Move the folder into:
xampp/htdocs/

### 3️⃣ Start XAMPP

Open **XAMPP Control Panel** and start:

- Apache
- MySQL

### 4️⃣ Setup Database

1. Open **phpMyAdmin**
2. Create a database:
elearning_platform

3. Import the SQL file.

### 5️⃣ Run the project

Open your browser and visit:
http://localhost/elearningplatform
